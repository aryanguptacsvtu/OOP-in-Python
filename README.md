# 🐍 OOP in Python

A hands-on collection of Python scripts exploring the core concepts of **Object-Oriented Programming (OOP)** -- from the basics to more advanced features.

---

## 📁 What's Inside

| File | Topic |
|---|---|
| `Class&Object.py` | Defining classes and creating objects |
| `Constructors.py` | `__init__` and constructor usage |
| `Instance&Class_Var.py` | Instance vs class variables |
| `Access_Modifiers.py` | Public, protected, and private members |
| `Getters_Setters.py` | Using `@property` for getters and setters |
| `ClassMethods1/2.py` | `@classmethod` and its applications |
| `StaticMethod.py` | `@staticmethod` usage |
| `Inheritance1–6.py` | Single, multiple, multilevel, and hybrid inheritance |
| `MethodOverriding.py` | Overriding methods in subclasses |
| `Super_a/b.py` | Using `super()` to call parent methods |
| `OperatorOverloading.py` | Overloading operators with magic methods |
| `MagicMethods.py` | Dunder methods (`__str__`, `__len__`, etc.) |
| `Decorators.py` | Function and class decorators |
| `Generators.py` | Generator functions with `yield` |
| `FunctionCaching.py` | Caching with `functools.lru_cache` |
| `Walrus.py` | The walrus operator (`:=`) |
| `dir_dict_help.py` | Exploring objects with `dir()`, `__dict__`, `help()` |
| `EXPERIMENTS/` | Miscellaneous experiments and scratch files |
| `Unit 1/` | Foundational unit-wise notes and examples |

---

## 🚀 Getting Started

**Prerequisites:** Python 3.x

```bash
# Clone the repository
git clone https://github.com/aryanguptacsvtu/OOP-in-Python.git
cd OOP-in-Python

# Run any script
python Inheritance1.py
```
